The **Foliage Interaction System** is a standalone plugin that adds harvesting, actor conversion, and respawn support for Unreal Engine foliage.\
It **requires the** Ultimate Multiplayer Inventory’s interaction system. This is built directly for my products to be plug-in play.

This plugin lets you turn static foliage instances into fully interactive gameplay elements such as stones, herbs, sticks, trees, ore nodes, bushes, and more.\
It is optimized for multiplayer projects and designed to be fully data-driven and Blueprint-friendly.

---

## **Key Features**

- Add on plugin (requires Ultimate Multiplayer Inventory)
- Supports both _Simple_ and _Complex_ foliage types
- Per-instance detection using UE foliage components
- Optional conversion of foliage instances into Actors
- Integration with UMI loot systems
- Integration with Respawn Manager Subsystem
- Multiplayer-aware harvesting and destruction flow
- Simple setup with a centralized Foliage Manager asset
- Fully Blueprint extendable
