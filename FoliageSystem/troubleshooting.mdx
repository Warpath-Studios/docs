description: "Nothing happens when I interact with foliage"
---

**Nothing happens when I interact with foliage**

- Check that your foliage mesh has a Foliage Type ID
- Ensure the character has a UMI Interaction Component
- Verify the Foliage Manager asset is referenced in the game mode or subsystem

**Actor spawns with incorrect rotation**

- Adjust mesh pivot
- Verify transform passed during conversion

**Foliage regrows instantly**

- Check respawn time in the Foliage Manager
