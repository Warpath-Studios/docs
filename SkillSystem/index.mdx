*A flexible, data-driven skill progression framework for Unreal Engine.*

Welcome to the **Universal Skill System**, a production-ready plugin designed to handle **any form of character progression** — from passive “use-based” skills to XP-driven leveling, multi-branch talent trees, parent/child skill ecosystems, and profession-style crafting skills.

UMSS is built to be:

- **Modular** — enable only the skill systems you need
- **Game-agnostic** — works with RPGs, survival games, MMOs, action games, and shooters
- **Data-driven** — all skills, XP rules, and trees are defined through Primary Data Assets
- **Blueprint-friendly** — no C++ required to use or extend
- **Multiplayer ready** — replicated out of the box

The plugin is designed for both beginners and advanced developers. You can drop it into an empty project or integrate it into a complex existing framework.
